Разработать основной экран приложения для учета личных финансов с использованием основных макетов и элементов
управления MAUI.
Примерный пошаговый гайд:
1. Создание нового проекта MAUI MyFinanceApp
2. Настройка главной страницы (MainPage.xaml) - замените содержимое MainPage.xaml на свой макет
3. Создание макета с использованием Grid
•
Разделите экран на 3 основные области:
• •
Верхняя: заголовок и общий баланс (20% высоты)
Центральная: список последних операций (60% высоты)
• Нижняя: кнопки действий (20% высоты)
4. Добавление элементов управления в верхнюю область
•
Label с текстом "Мои финансы"
•
Label с суммой общего баланса (например, "25 000 р")
•
ProgressBar для визуализации прогресса
5. Pеализация центральной области с CollectionView
•
Создайте CollectionView для отображения списка транзакций
•
Добавьте 5+ тестовых транзакций с разными категориями
•
Для каждой транзакции отображайте: иконку категории, описание, сумму, дату
6. Создание нижней панели с HorizontalStackLayout - разместите 3 кнопки "Доход", "Расход", "Отчеты".
Топ советы:
1. Начните с планирования - прикиньте схему макета до разработки.
2. Тестируйте часто - запускайте приложение после добавления каждого major элемента.
3. Используйте документацию - Microsoft Learn: Макеты в MAUI
4. Не усложняйте - сосредоточьтесь на базовых элементах, кастомизацию оставьте на следующие занятия.
